# Agent Instructions

This file follows the open [agents.md](https://agents.md) standard and is
read by Cursor, Aider, OpenAI Codex, Gemini CLI, Jules and other AI
coding agents that look for `AGENTS.md`.

The complete instructions (TIA Portal Openness API reference, SimaticML
XML conventions, the 17 `tia_*` Language Model Tools, build commands,
autonomous compile-fix loop) are maintained in a single source of truth
shared with GitHub Copilot:

> See [.github/copilot-instructions.md](.github/copilot-instructions.md).

## Code style

- Edit `.xml` / `.scl` / `.s7dcl` / `.s7res` files using the rules from
  [.github/copilot-instructions.md](.github/copilot-instructions.md).
- Do not hand-edit `.s7dcl` blocks as SCL — they are LAD/FBD source.
- Do not modify files under `.tia-cache/` or `TiaExport/` — those are
  regenerated by the extension.

## Pull request guidelines

- Keep changes scoped to the requested task. Avoid drive-by reformatting
  of TIA-generated files, the line-by-line diff is what the bridge uses
  to decide what to push back into TIA Portal.
- Always run `tia_compile` (or `npm run compile` for the extension
  itself) before submitting.
